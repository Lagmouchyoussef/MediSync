# MediSync - Instructions de démarrage

## Démarrage du Backend (Django)
1. Ouvrez un terminal dans le dossier `MediSync/backend`
2. Activez l'environnement virtuel : `venv\Scripts\activate` (Windows)
3. Lancez le serveur : `python manage.py runserver 8001`

## Démarrage du Frontend (React)
1. Ouvrez un autre terminal dans le dossier `MediSync/frontend`
2. Lancez le serveur : `npm start`

## Test de la connexion
- Frontend : http://localhost:3000
- Backend API : http://localhost:8001/api/health/
